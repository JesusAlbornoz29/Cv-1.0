# **App Name**: PortfolioPro

## Core Features:

- Importación y Análisis de CV: Importa y analiza datos de CV existentes desde una URL proporcionada (ej., GitHub, Netlify). La aplicación extraerá automáticamente la información relevante utilizando herramientas de análisis para completar el portafolio.
- Herramienta de Mejora de CV con IA: Una función de IA generativa que revisa los datos del CV importado y proporciona sugerencias de mejora. Analiza el contenido, el formato y las palabras clave para sugerir mejoras en cuanto a claridad, impacto y relevancia. La IA actúa como una 'herramienta' para brindar recomendaciones personalizadas.
- Exhibición de Proyectos: Sección dedicada a exhibir proyectos terminados con enlaces a la demostración en vivo y una imagen de vista previa.
- Diseño Reactivo: Asegura que el sitio web del portafolio sea totalmente reactivo y se adapte a varios tamaños de pantalla y dispositivos, proporcionando una experiencia de usuario fluida en computadoras de escritorio, tabletas y teléfonos móviles. Construido con React.
- Actualizaciones Dinámicas de Contenido: Implementa un sistema para actualizar fácilmente el contenido del portafolio (proyectos, habilidades, experiencia) sin necesidad de modificar el código.

## Style Guidelines:

- Color primario: Morado intenso (#673AB7) para transmitir creatividad y sofisticación.
- Color de fondo: Gris muy claro (#F5F5F5) para proporcionar un telón de fondo limpio y moderno.
- Color de acento: Violeta eléctrico (#8200b9) para resaltar elementos clave y llamadas a la acción.
- Fuente del encabezado: 'Space Grotesk', sans-serif, para una sensación contemporánea y técnica.
- Fuente del cuerpo: 'Inter', sans-serif, para legibilidad y una apariencia neutral.
- Usa un conjunto de iconos modernos y minimalistas para representar diferentes habilidades y tecnologías.
- Implementa un diseño limpio y modular con una jerarquía visual clara y un amplio espacio en blanco.
- Animaciones y transiciones sutiles para mejorar la participación del usuario y crear una sensación de pulido.
